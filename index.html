Projects

Links - Bookmarking App

HTML SASS JavaScript Python
Django PostgreSQL jQuery Bootstrap

An app which allows users to save the URL's of any website for easy reference. Built using the Django framework and hosted on Heroku, users can add, edit, delete, and organise their bookmark collection using a simple and intuitive UI.

The app has built in scraping to automatically pull site metadata and icons which makes getting detailed information for the URL as simple as clicking one button.

There is also a dedicated Chrome Browser extension that makes adding bookmarks even easier.

 

Cheers - The Drinks DB

HTML CSS JavaScript Python
Flask MongoDB jQuery Bootstrap

This app was built using the Flask micro web framework and then hosted with Heroku.

Users can add, edit, and delete their own drinks. Drinks submitted by other users can be commented on and added to a list of favorites.

There are extensive search, sort, and filtering options available to make finding what you want quick and easy.
